# 数值计算算法 (Numerical Calculation Algorithms)

本目录包含各种数值分析方法的 Python 实现，涵盖逼近、积分、线性代数和非线性方程求解。

## 文件说明

### 1. 函数逼近 (`work2.py`)
- **目标**: 比较不同的函数逼近方法。
- **方法**:
  - **最佳平方逼近**: 使用勒让德正交多项式。
  - **最小二乘拟合**: 标准多项式曲线拟合。
- **输出**: 比较逼近方法与真实函数的直观图表和误差分析。

### 2. 数值积分 (`work3.py`)
- **目标**: 高精度地数值计算定积分。
- **方法**:
  - **复化梯形公式**: 基础数值积分方法。
  - **龙贝格积分 (Romberg Integration)**: 一种迭代方法，利用 Richardson 外推法改进梯形公式以获得更高的精度。
- **输出**: 详细的迭代步骤、收敛率和效率比较。

### 3. 线性方程组直接解法 (`work4.py`)
- **目标**: 求解线性方程组 ($Ax = b$)。
- **方法**:
  - **列主元高斯消元法**: 数值稳定的消元方法。
  - **LU 分解 (Doolittle 方法)**: 将矩阵 $A$ 分解为下三角 ($L$) 和上三角 ($U$) 矩阵。
- **输出**: 逐步矩阵变换和解向量。

### 4. 线性方程组迭代解法 (`work5.py`)
- **目标**: 迭代求解大型线性方程组。
- **方法**:
  - **高斯-塞德尔迭代 (Gauss-Seidel)**: 标准迭代求解器。
  - **逐次超松弛迭代 (SOR)**: 加速迭代方法。包含寻找最佳松弛因子 $\omega$ 的逻辑。
- **输出**: 迭代次数、收敛性检查和残差分析。

### 5. 非线性方程求解 (`work6.py`)
- **目标**: 寻找非线性方程 ($f(x) = 0$) 的根。
- **方法**:
  - **不动点迭代**: 基础迭代方法 $x_{k+1} = \phi(x_k)$。
  - **Steffensen 方法**: 不动点迭代的加速技术（实现二次收敛）。
  - **牛顿法 (Newton's Method)**: 利用导数实现快速收敛（二次收敛）。
  - **二分法**: 稳健的区间搜索求根方法。
- **输出**: 收敛路径、迭代次数和计算成本比较。

## 用法
每个文件都是独立的 Python 脚本。您可以直接运行它们：
```bash
python work2.py
```
大多数脚本都包含 `main` 函数，带有示例测试用例或用于输入的交互式提示。
