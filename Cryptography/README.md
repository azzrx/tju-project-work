# 密码学算法 (Cryptography Algorithms)

本目录包含各种密码学算法和数论概念的 C++ 实现，主要侧重于 RSA 算法和模运算优化。

## 文件说明

### 1. RSA 加密与解密 (`work1.cpp`)
- **功能**: 实现标准的 RSA 加密和解密流程。
- **主要特性**:
  - 密钥生成（根据 `e` 和 `phi` 计算私钥 `d`）。
  - 用于加密和解密的模幂运算。
  - 命令行接口，支持使用自定义参数执行加密或解密操作。

### 2. RSA 小加密指数攻击 (`work2.cpp`)
- **功能**: 演示当多个接收者使用相同的小加密指数（$e=3$）时，如何攻击 RSA 系统。
- **方法**:
  - 使用中国剩余定理 (CRT) 组合截获的密文。
  - 计算组合结果的整数立方根以恢复原始消息。

### 3. 强素数生成 (`work3.cpp`)
- **功能**: 生成用于密码学的“强素数”。
- **详情**:
  - 确保生成的素数 $p$ 具有特定属性：
    - $p-1$ 有一个大素数因子 $r$。
    - $p+1$ 有一个大素数因子 $s$。
    - $r-1$ 有一个大素数因子 $t$。
  - 这些属性有助于抵御某些因数分解算法（例如 Pollard 的 p-1 算法）。

### 4. K-ary 分解 (`work4.cpp`)
- **功能**: 实现指数分解的 $k$-ary 窗口方法。
- **应用**: 用于加速模幂运算，通过一次处理指数的 $k$ 位来减少乘法次数。

### 5. wNAF 编码 (`work5.cpp`)
- **功能**: 实现窗口非相邻形式 (wNAF) 编码算法。
- **应用**:
  - 将整数转换为使用有符号数字的表示形式。
  - 最小化非零数字的数量（汉明权重），这显着优化了椭圆曲线标量乘法和模幂运算。

### 6. 高效模幂运算与 CRT 解密 (`work6.cpp`)
- **功能**: 实现模运算的高级优化技术。
- **算法**:
  - **Shamir 技巧**: 高效计算 $g^a h^b \pmod p$。
  - **扩展 Shamir 技巧**: 使用 NAF 的优化版本。
  - **CRT RSA 解密**: 通过分别计算模 $p$ 和模 $q$ 然后合并结果来加速 RSA 解密。

## 用法
大多数程序都是基于命令行的。请使用 C++ 编译器（如 g++ 或 MSVC）编译它们并运行生成的可执行文件。某些文件（如 `work1.cpp`）接受命令行参数。
